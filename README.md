# 🗳️ Election Assistant

## 📌 Problem Statement
This project helps users understand the election process, timelines, and steps in an interactive way.

## 🚀 Features
- Step-by-step election guide
- Election timeline visualization
- Chatbot for answering questions
- User-friendly interface

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS

## ▶️ How to Run
1. Install Flask:
   pip install flask
2. Run:
   python app.py
3. Open:
   http://127.0.0.1:5000/
